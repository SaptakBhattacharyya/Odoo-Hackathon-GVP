# FleetMetrics — Troubleshooting & Best Practices

## Common Issues

### Vehicle won't change status
Make sure the vehicle doesn't have active trips. A vehicle with an active trip cannot be retired or manually set to available. Complete or cancel all trips first.

### Driver can't be assigned
Check that the driver's status is "On Duty." Drivers on break, busy, or suspended cannot be assigned new trips. Also verify their license class matches the vehicle type requirements.

### Expense rejected without reason
All rejections must include a reason. If you see a rejection without explanation, contact your manager.

### Trip stuck in "Dispatched"
The dispatcher or driver needs to update the trip to "In Transit" when the vehicle departs. This is not automatic.

### Maintenance costs not showing in expenses
Maintenance costs are tracked separately from trip expenses. Check the Maintenance page for repair costs and the Expenses page for trip-related costs.

## Best Practices

### Fleet Optimization
1. Regularly review vehicle utilization rates — retire underused vehicles
2. Schedule preventive maintenance to avoid costly breakdowns
3. Monitor fuel consumption trends to identify inefficient routes
4. Match vehicle capacity to cargo requirements to minimize waste

### Driver Management
1. Review safety scores weekly and address declining trends
2. Rotate drivers between routes to prevent fatigue
3. Track complaint patterns and address systemic issues
4. Ensure license expiry dates are monitored proactively

### Cost Control
1. Categorize all expenses properly for accurate reporting
2. Set up approval workflows to catch unusual charges
3. Compare fuel costs across drivers and vehicles to find outliers
4. Track maintenance costs per vehicle to identify money pits

### Trip Efficiency
1. Plan routes to minimize empty return trips
2. Consolidate cargo when possible to reduce trip count
3. Track estimated vs actual fuel to improve forecasting
4. Monitor on-time delivery rates for service quality

## System Tips

### Keyboard Shortcuts
- Use the search bar to quickly find vehicles, drivers, or trips
- Filter tables by status to focus on actionable items

### Data Export
- Analytics data can be exported for external reporting
- Expense reports can be generated by date range, category, or vehicle

### Security
- Always log out when leaving the terminal
- Report suspicious activity to your manager
- Do not share login credentials
